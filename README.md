# PDF 텍스트 추출 프로그램

PDF 파일에서 텍스트를 추출하는 웹 애플리케이션입니다.

## 기능

- PDF 파일 업로드
- PDF에서 텍스트 자동 추출
- 추출된 텍스트 확인 및 다운로드

## 설치 방법

```bash
npm install
```

## 실행 방법

```bash
npm run dev
```

브라우저에서 `http://localhost:3000`으로 접속하세요.

## 사용 방법

1. "PDF 파일 선택" 버튼을 클릭하여 PDF 파일을 선택합니다.
2. "텍스트 추출하기" 버튼을 클릭합니다.
3. 추출된 텍스트가 화면에 표시됩니다.
4. "텍스트 다운로드" 버튼을 클릭하여 텍스트 파일로 저장할 수 있습니다.
